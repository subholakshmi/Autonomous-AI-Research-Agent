# 🔍 Autonomous AI Research Agent

An AI-powered system that performs end-to-end research on a given topic using a multi-step workflow. The system generates search queries, retrieves relevant information, summarizes content, and produces a structured research report.

## 🚀 Features
- Automated research pipeline
- Query generation using LLMs
- Content summarization
- Structured report generation
- Streamlit-based interactive UI

## 🧠 How It Works
1. User inputs a topic
2. AI generates search queries
3. System retrieves relevant content (simulated)
4. AI summarizes the content
5. AI generates a structured research report

## 🛠️ Tech Stack
- Python
- Groq API (LLaMA 3)
- Streamlit
- Prompt Engineering
